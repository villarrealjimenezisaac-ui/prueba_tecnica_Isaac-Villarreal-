# Prueba Técnica – Data Analyst Retail

## Descripción

Este repositorio contiene mi solución para la prueba técnica de Data Analyst para una cadena de retail multiformato con presencia en Centroamérica.

El análisis fue desarrollado utilizando SQL, Python, modelado dimensional y principios de analítica de negocio para responder los requerimientos planteados en los bloques de auditoría de datos, análisis avanzado, modelado, experimentación, diseño de KPIs y presentación ejecutiva.
